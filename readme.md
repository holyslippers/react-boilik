# React Boilerplate 

## Версии пакетов

- React 16.9.x
- Typescript 3.8.3
- Webpack 4.43
- Babel 7.9.6 
- StyledComponents v4.4.1
- AntDesign 4.2.2

## Запуск dev среды 

npm start

## Билд

npm run build

## Конфигурация

- webpack.dev.config.js - dev среда
- webpack.prod.config.js - prod среда

## Команды 

- `npm run lint` Запуск eslint 
- `npm run lint:fix` Запуск фиксов eslint 
- `npm run test` Запуск тестов через Mocha
